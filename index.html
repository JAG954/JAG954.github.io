Production Scheduling · Capstone

Boeing 787 FGI Scheduling Model

Purdue IE capstone modeling Boeing 787 movement through Final Goods Inventory (FGI) and delivery operations. I led Python development and version control for a five-person team, and the project placed third among Purdue IE capstone teams. The model represents aircraft state, location occupancy, movement feasibility, labor-driven task completion, paint and inspection workflows, queue behavior, and delivery readiness under manufacturing constraints.

Python, Jupyter, modular package code, staged Excel input pipelines, move-time matrices, daily trace exports, labor allocation outputs, KPI tables, higher-rate scenario simulations through May 2027.

Operations Research

LP Solver Implementations

From-scratch Python implementations of two-phase primal Simplex and primal-dual Interior Point Method solvers, including degeneracy handling and anti-cycling safeguards. Built as part of an optimization-for-hydrological-modeling technical project, the repository pairs reusable LP model builders with structured test cases, pytest coverage, and benchmark verification against Gurobi to validate objective values, infeasible/unbounded behavior, and commercial-solver agreement.

Python, NumPy, Jupyter, two-phase Simplex, primal-dual IPM, pytest, Gurobi verification, LP test harness.

View repository

Public-source document corpus on lead poisoning in Milwaukee, built in collaboration with Dr. Elizabeth Hoffman at Purdue University through the Margo Katherine Wilke Undergraduate Research Internship Program.
